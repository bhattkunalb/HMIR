API Endpoints

HMIR serves an OpenAI-compatible API. Use these endpoints in your tools:

Chat: http://localhost:8080/v1/chat/completions
Health: http://localhost:8080/v1/health
Telemetry: http://localhost:8080/v1/telemetry
Cursor / VS Code Integration

Add this to your editor settings to use HMIR as a local AI backend:

"openai.apiBase": "http://localhost:8080/v1"
"openai.model": "hmir-local"
CLI Commands
$ hmir start — Start API + Dashboard
$ hmir stop — Stop all services
$ hmir dashboard — Open native dashboard
$ hmir download <model> — Download a model via CLI
